# Privacy Policy

## Your Privacy Matters

This website is built with privacy in mind.

## What We Collect

### Website Analytics
This site currently does not use any analytics. If added in the future, only privacy-friendly analytics (like Fathom Analytics) will be used.

### Comments (Giscus)
Comments are powered by GitHub Discussions via Giscus. When you comment:
- You sign in with your GitHub account
- Your comment is stored in this repository's GitHub Discussions
- GitHub's privacy policy applies: [https://docs.github.com/en/site-policy/privacy-policies](https://docs.github.com/en/site-policy/privacy-policies)

## Third-Party Services

### GitHub Pages
This site is hosted on GitHub Pages. GitHub may collect technical information such as IP addresses. See [GitHub's privacy statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

### External Links
This site contains links to external sites (QRZ.com, ARRL.org, etc.). We are not responsible for the privacy practices of these sites.

## Your Rights

You have the right to:
- Access your data (via your GitHub account for comments)
- Request deletion of your comments (delete via GitHub or contact us)
- Opt out of commenting (just don't comment!)
