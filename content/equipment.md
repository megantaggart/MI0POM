# Station Equipment

This page documents the equipment in my shack. Most antennas are homemade!

## Transceivers

### HF Station
**Flex Radio 6400**
- Software Defined Radio (SDR)
- Bands: 160m - 6m
- Remote operation capable
- Paired with Flex Tuner Genius

### VHF/UHF
**Yaesu FT-991A**
- All-mode transceiver
- Bands: HF/6m/2m/70cm
- 100W on HF, 50W on VHF/UHF
- Built-in antenna tuner

## Antenna Tuners

- **Flex Tuner Genius** - Automatic tuner for the Flex 6400
- **MFJ Antenna Tuner** - Automatic tuner for the Yaesu

## Antennas

### HF

**1/4 Wave Vertical - Homemade**
- Bands: 80m / 60m / 40m / 30m / 20m / 15m / 10m
- Multiband vertical antenna
- Homemade construction

**Hex Beam - Homemade**
- Bands: 20m / 17m / 15m / 12m / 10m / 6m
- Lightweight directional beam
- Yaesu rotator with Arduino controller
- Full homemade build including controller

**160m Sky Loop - Homemade**
- Band: 160m (top band)
- Large loop antenna
- Homemade construction

**20m Dipole - Homemade**
- See [project page](/projects/20m-dipole/) for build details

**2m Ground Plane - Homemade**
- Simple vertical for local VHF work

## Software & Servers

**OpenWebRX**
- Web-based SDR receiver server
- Running on HackRF One SDR
- Remote monitoring and reception
- Accessible from anywhere
